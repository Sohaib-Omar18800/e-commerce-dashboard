# 📊 E-Commerce Sales Performance Dashboard (Advanced Excel)

An interactive corporate-grade dashboard built entirely in Microsoft Excel, leveraging **Power Query** for advanced ETL processes and **Power Pivot** for professional data modeling to analyze US marketplace distributions and tracking sales parameters from 2021 to 2022.

![E-Commerce Sales](ecommerce%20sales%20dashboard.png)

## 🎯 Project Objectives & Insights
* **Sales & Order Tracking:** Analyzed a total revenue of $23.1M across 113,270 orders, uncovering October as the highest-performing month.
* **Geographical Distribution:** Built a dynamic US Sales Map visual to isolate top-performing states and regions for targeted inventory allocation.
* **Shipping & Operations:** Monitored delivery health statuses (Late Deliveries vs. Advance/On-time Shipping) to identify operational bottlenecks.

## 🛠️ Tech Stack & Methods
* **Data Ingestion & ETL:** Power Query (Data cleaning, merging, and type standardization).
* **Data Modeling:** Power Pivot (Star Schema configuration, setting up relationships).
* **Analytical Computations:** DAX Measures (Calculating custom metrics, MoM Sales Analysis, and growth trends).
* **Interface Design:** Excel Pivot Tables, Slicers, and advanced charting layouts.
